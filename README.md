POCKET PET

PocketPet is a virtual pet for mobile android devices inspired by desktop companions like BonziBuddy or Desktop Mate.
The per reacts to your actions and your phone's environment, follows you across apps through an overlay, and can be customized.

FEATURES

Interactive Pet — Reacts to screen taps, and reacts to phone orientation via the phone's accelerometer.
Physics Movement — Drag and drop pet manually. Pet moves through simulated gravity.
Overylay Mode — The pet can 'leave' the app as an overlay and follow the user across other apps.
Customization — You can choose a pet type and change the pet name.

INSTALLATION

1. Clone the repository: git clone https://github.com/Cruces423/PocketPet01.git
2. Open the project in Android Studio.
3. Build or run on an android device or emulator.
(Some features require real device sensors such as the accelerometer.)
